# circuitfolio

A minimalist portfolio site for EE students that auto-syncs and showcases GitHub projects with circuit-themed styling

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```

## Features

- GitHub API integration to auto-fetch and display repositories with stats
- Circuit board inspired UI with trace-line animations and component-styled cards
- Filterable project grid by language, topic, or custom tags like 'embedded' or 'power-systems'
